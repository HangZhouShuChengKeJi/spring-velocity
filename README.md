# spring-velocity
spring velocity 2.2 版本支持

# 使用
```xml
<dependency>
    <groupId>org.yg.spring</groupId>
    <artifactId>springframework-velocity</artifactId>
    <version>0.0.0.1</version>
</dependency>
```
